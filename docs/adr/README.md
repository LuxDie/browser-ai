# Architecture Decision Records (ADRs)

Mantén un registro versionado de decisiones de arquitectura.

## Convenciones
- Formato: `NNNN-titulo-decision.md`
- Idioma: castellano
- Contenido mínimo: contexto, decisión, consecuencias

## Plantilla
```
# Título

## Contexto

## Decisión

## Consecuencias
```
